# LeadMarketing — Sprint Roadmap
**Goal: get more consulting clients at low cost via consulting.levelcoding.com**

---

## Sprint 1 — Reply Tracking & Follow-up Sequences
*The fastest way to close more clients is to follow up. Most deals happen on touch 3–5.*

- [ ] 1.1 Add a "Replied" status and track the reply date so you can measure email → reply rate per country/domain
- [ ] 1.2 Build a follow-up queue: leads emailed 5–7 days ago with no reply surface in a new "Follow Up Today" tab
- [ ] 1.3 Generate a shorter follow-up email template (2 paragraphs, references prior email, asks for 15-min call)
- [ ] 1.4 Log every follow-up attempt so you can see how many touches it took to get each reply

---

## Sprint 2 — Conversion Funnel & What's Working Analytics
*You can't optimize what you can't measure. Know which countries, domains, and templates convert.*

- [ ] 2.1 Add a Conversion Funnel card on the dashboard: New → Emailed → Replied → Call Booked → Closed (with %)
- [ ] 2.2 Show reply rate by country so you know where to double down (e.g. Switzerland > Denmark)
- [ ] 2.3 Show reply rate by domain type (CRM, no-website, health, B2B, tourism) to identify best-fit verticals
- [ ] 2.4 Add a "best segment this week" highlight card that auto-picks the top-performing country+domain combo

---

## Sprint 3 — WhatsApp Multi-Channel Outreach
*Emails get ignored. WhatsApp gets read. For no-website leads (local shops) this is the highest-conversion channel.*

- [ ] 3.1 Build a WhatsApp queue tab: no-website leads with phone, sorted by country, with "Sent" / "No Answer" / "Replied" tracking
- [ ] 3.2 Generate a short WhatsApp message template per lead (3 sentences max, ends with booking link)
- [ ] 3.3 Add a "Multi-touch" indicator on each lead showing which channels were used (Email ✓, WhatsApp ✓)
- [ ] 3.4 Show WhatsApp reply rate vs email reply rate side-by-side to see which works better per country

---

## Sprint 4 — Lead Scoring & Priority Queue
*Stop treating all 3,000 leads equally. High-value targets deserve more touches and better copy.*

- [ ] 4.1 Auto-score each lead on import: domain age, has phone, has website, business category (0–10 score)
- [ ] 4.2 Add a "High Value" filter on the today grid — show top-scored leads first, not alphabetical
- [ ] 4.3 Build a "Hot Leads" dashboard card: leads with high score + no email sent yet = immediate revenue opportunity
- [ ] 4.4 Allow manually boosting a lead score with a star/pin so it stays at top of every future batch

---

## Sprint 5 — Booking Link & Response Automation
*Every email ends with "schedule a call." Make that link trackable and reduce friction to close.*

- [ ] 5.1 Add UTM tracking to `consulting.levelcoding.com/book/3` links in generated emails (utm_source=leadmanager&utm_country=CH)
- [ ] 5.2 Add a "Call Booked" status and log the booked date — this is your real conversion metric
- [ ] 5.3 Show a "Calls Booked This Month" counter on the dashboard so you see ROI from outreach instantly
- [ ] 5.4 When a lead replies, surface a quick "Mark as Call Booked" button in the lead row (no need to open detail page)

---

## Sprint 6 — Referral & Warm Lead Acceleration
*A referred lead closes 3× faster. Build a way to track and prioritize intros.*

- [ ] 6.1 Add a "Source" field per lead: Cold Outreach / LinkedIn / Referral / Inbound — track where wins come from
- [ ] 6.2 Build a "Warm Leads" tab that shows referral-source leads and inbound contacts separately from cold pool
- [ ] 6.3 Add a note field on each closed/replied lead so you can record who referred them and follow up for more intros
- [ ] 6.4 Show a monthly Source Breakdown chart: which acquisition channel (email, WA, referral) brought the most clients

---

## Current Metrics to Track
| Metric | Target | Notes |
|---|---|---|
| Emails sent / day | 50 | Current warmup quota |
| Reply rate | > 3% | Industry avg is 1–2% |
| Calls booked / month | 10 | = ~2 clients/month |
| WhatsApp reply rate | > 8% | Higher than email |
| Cost per client | < €50 | Time + tool cost only |
